I released a new version of my iTunes Export Scala utility.<br /><br />This is a port of the original .Net application to Scala.  <a href="http://www.scala-lang.org/">Scala</a> is a hybrid Object Oriented/Functional language that compiles to Java class files and executes on the JVM.  This port enables Mac OS X users to access iTunes Export features. The original iTunes Export application was written in .Net and does not run on Mac OS X.<br /><br />iTunes Export exports your iTunes playlists as M3U or WPL files, allowing you to setup playlists in iTunes and use them with other software or devices.<br /><br />This release adds support for Extended M3U, WPL, and ZPL playlist formats.  It also provides the ability to override the default (or arbitrary) path prefix for the music files.<br /><br />The project home page is here: <a href="http://www.ericdaugherty.com/dev/itunesexport/scala/">http://www.ericdaugherty.com/dev/itunesexport/scala/</a><br /><br />The project is hosted at Google Code.  You can check out the <a href="http://code.google.com/p/itunesexport-scala/">Google Code project home page</a> if you want to browse the source tree or track issues, etc.<br /><br />I have received several new feature requests and bug reports recently and I plan on getting to them soon. Don't be afraid to ask for new features (eric@ericdaugherty.com).
