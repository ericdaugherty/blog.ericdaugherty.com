I am launching a new project: <a href="http://www.ericdaugherty.com/dev/itunesexport/scala/">iTunesExport Scala</a>.  This is a port of the original .Net application to Scala.  <a href="http://www.scala-lang.org">Scala</a> is a hybrid Object Oriented/Functional language that compiles to Java class files and executes on the JVM. <br /><br />I am launching this port to enable Mac OS X users to access iTunes Export features.  The original iTunes Export application was written in .Net and does not run on Mac OS X.<br /><br />The project home page is here: <a href="http://www.ericdaugherty.com/dev/itunesexport/scala/">http://www.ericdaugherty.com/dev/itunesexport/scala/</a><br /><br />The project is hosted at Google Code.  You can check out the <a href="http://code.google.com/p/itunesexport-scala/">Google Code project home page</a> if you want to browse the source tree or track issues, etc.<br /><br />This is my first Scala application so I would appreciate any feedback on the source code.  My goal is to continue to add features until parity is reached between this and the original.
