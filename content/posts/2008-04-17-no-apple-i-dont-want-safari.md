It is old new by now, but there was a bit of hubbub recently about Apple's inclusion of Safari 3 in its update manager (for Windows).<br /><br />Gruber has a good <a href="http://daringfireball.net/2008/03/update">writeup</a>, concluding that there would be no issue at all if Apple left the box unchecked by default or visually separated new installs.  I pretty much agree.  I don't really have a problem with Apple using the update tool as a way to advertise new applications, but don't make it a default install.  However, it doesn't end there.<br /><br />Apple KEEPS TRYING TO INSTALL IT!  I have two browsers already (Firefox, IE), I hardly need a third.
