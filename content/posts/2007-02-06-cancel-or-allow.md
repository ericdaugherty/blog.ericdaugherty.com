OK, in case you think I'm an Apple basher because of my <a href="http://www.ericdaugherty.com/blog/2007/02/no-iphone-for-me.html">iPhone post</a>, I thought I'd pass along the new 'Get a Mac' commercial.  The newest commercial, entitled <a href="http://www.apple.com/getamac/">Security</a>, depicts the new Vista security enhancements, which prompt users asking if they want to allow a program to execute an action that may be risky.  While this is a good concept, apparently it is a little too aggressive in Vista <a href="http://www.schneier.com/blog/archives/2006/04/microsoft_vista.html">based on what I've read</a>.<br /><br />Either way, it is a great commercial.
