I've been using Excel 2007 for a while now, and while I generally like the 'ribbon' and new layout, I've had a chronic issue with crashes.<br /><br />When I scroll using my ThinkPad TrackPoint scroll feature, Excel crashes.  For quite a while I assumed this was a driver/hardware issue related to the TrackPoint driver.  Today I discovered a coworker has the same issue, using the scroll wheel on his mouse. <br /><br />After doing a bit of Googling, it seems <a href="http://www.sidesofmarch.com/index.php/archive/2007/06/05/excel-crashes-and-flooding-microsoft-with-error-reports/">others</a> are seeing this as well.  Unfortunately, I haven't found a solution yet.  I can't believe a feature as basic as this is causing widespread crashes without a fix.  If anyone has a solution, please let me know.
