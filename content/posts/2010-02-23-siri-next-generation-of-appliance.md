In my <a href="http://blog.ericdaugherty.com/2010/02/why-ipad-will-succeed-and-rise-of.html">previous post</a>, I discussed the trend toward computing appliances (ie DVRs, Kindles, etc.) instead of general purpose computers.&nbsp; On the recommendation of <a href="http://www.merlinmann.com/">Merlin Mann</a> on <a href="http://twit.tv/mbw">MacBreak Weekly</a>, I downloaded the <a href="http://siri.com/">Siri iPhone application</a> and gave it a try.&nbsp; Wow.

The Siri application attempts to be the ubiquitous Star Trek computer.&nbsp; Just ask it a question and it will give you the answer.&nbsp; It provides both voice and text interaction modes, and an easy user interface that exposes common feature easily.

It won't do everything you want, and I'm not sure this specific application will be something I use regularly, but at a minimum it provides an interesting example of where the world is going.&nbsp; Imagine using this with what I imagine will become ubiquitous Bluetooth headsets.&nbsp; Just ask a question and have the answer spoken to you.&nbsp; We are not there yet, but we're starting to get really close.

This is an example of what I consider the appliance trend, applied to software.&nbsp; The application provides easy access to a common (if limited) set of features that are intuitive to use.
