As I outlined in my <a href="http://www.ericdaugherty.com/blog/2007/12/new-phone.html">previous post</a>, I'm using a new 755p.  To get my 755p in prime form, there are a bunch of tweaks and programs to install right out of the box.<br /><br />First, there is some software that comes with the device but isn't shipped on the CD.  It can be downloaded <a href="https://ws.palm.com/mypalm/MyPalmGenericUser/ControllerGeneric.jsp?&amp;action=showbonus&amp;productName=TREO755P">here</a>.  Some of the applications do require you to enter your serial number.  The software I downloaded was:<br /><ul><li>Versamail - needed for Outlook ActiveSync</li><li>Voice Dialing - I have found this only marginally useful, especially with the Treo's ease of use and it's so-so accuracy<br /></li><li>Bejeweled - addictive Game<br /></li><li>Solitaire - obvious</li><li>Palm Files - So you can browse files on the SD card</li></ul>After that, I grabbed:<br /><ul><li><a href="http://www.palm.com/us/support/jvm/download.html">IBM Java VM</a></li><li><a href="http://gmail.com/app/v1.1.0/L1/gmail-g.jar">Gmail Java Application</a> - Google didn't see to want me to d/l it using the mobile apps page, so the link there is direct to the jar file.  I also linked to 1.1.0 instead of 1.1.1 or 1.5.0 since 1.1.0 seemed to work the best.<br /></li><li><a href="http://www.google.com/gmm">Google Maps</a></li></ul>There are several configuration changes that I make right out of the box as well.<br /><br />Phone Display Preferences (General Preferences on older Treos):<br /><ul><li>Change 'Show DialPad' to 'Show Wallpaper' and select a background image.</li><li>Change 'Typing dials phone number' to 'Typing starts contact search'<br /></li><li>I set the Favorites Buttons Rows to 1.</li><li>I turn off the 'Blink green LED when in coverage' on the 650, but the 755 doesn't seem to have that setting.</li></ul>The 'Typing ...' setting is the key feature for the Treo in my eyes, and is what really makes it a great phone.  I can't believe this setting is set to 'dial number' by default, as it really hides the usability.  For the uninitiated, to call someone I turn the phone on, and then start typing their name on the keyboard.  You can also mix first and last names, so to dial myself (hey, why not) I would turn the phone on and type 'ed'.  If that doesn't narrow it down enough, you can keep typing, such as 'edau'.  For me, 2-3 letters finds just about everyone.
