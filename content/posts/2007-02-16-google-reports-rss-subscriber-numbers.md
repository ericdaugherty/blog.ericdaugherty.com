This is a bit of an '<a href="http://en.wikipedia.org/wiki/Inside_baseball">inside baseball</a>' post, but...<br /><br />Online feed readers, such as <a href="http://www.bloglines.com">Bloglines</a>, <a href="http://www.google.com/reader">Google Reader</a>, etc. allow many people to subscribe to a single feed without actually querying the feed for each user.  This means that tools such as <a href="http://www.feedburner.com">FeedBurner</a> are unable to track the number of subscribers accurately unless the tools report the number of readers.  Bloglines has done this for a while, but today Google <a href="http://googlereader.blogspot.com/2007/02/one-subscriber-two-subscribers-three.html">announced</a> that they are now reporting the number as well.  Finally. <br /><br />Feedburner <a href="http://blogs.feedburner.com/feedburner/archives/2007/02/the_google_effect.php">says</a> the number will be incorporated into their reports tonight, so tomorrow my number should double... to 2.  :)<br /><br />Google is a little late the the party here, but I'm glad they finally showed up.
